# material-suitability-audit

学习材料多角度适合度审计。五维度证据驱动评估 → 教学重构 → 用户确认修改。

## 为什么需要这个

你拿到一本教材、一篇笔记、一条学习路线——但你不知道它适不适合你的学习方式。

翻两页觉得"怪怪的"，但说不清哪里怪。硬着头皮学，越学越拧。

这个 skill 做的事：用五个维度诊断"哪里不合适"，然后产出**教学重构版**——保证原文信息不失真（定义、公式、推导、例子全在），按"直觉→推导→因果→自查"重组结构和表述方式，交给用户确认修改。

## 核心理念

> 不是"这个材料好不好"——是"这个材料对理解型学习者好不好"。

- 证据驱动，每条判断引用原文具体文本
- 五维度框架：框架先行 → 深度加工 → 主动建构 → 因果逻辑 → 信息节奏
- 优化方案：教学重构（信息不失真，按认知路径重组结构和表述）
- **两条路径**：标注增强（外围加 🔑💭✏️🔗）vs 教学重构（📖 体内段落翻转），根据原文"抽象→具体"程度选择
- **📖 体内翻转**：正文从抽象定义出发 → 翻转段落顺序为"具体直觉→抽象定义"；标注入口速查表

## 用法

```
评估一下这个教材的第2章适不适合我
```

```
帮我用五维度分析这条学习路线
```

```
这个笔记是我从别人那里拿的，从多角度看适合度如何
```

## 你会得到

```
# {材料名}适合度评估与优化方案

## 一、多角度适合度评估
  角度1-5 逐一展开，每项配证据表 + 原文引用

## 二、总评
  六维评分表 + 一句话结论

## 三、优化方案
  教学重构版（完整内容）
  改动说明 + 待用户确认（可修改）
```

## 适用场景

- 拿到新教材，不确定适不适合自学
- 别人推荐的学习路线，不知道合不合自己的学习风格
- 想优化现有学习材料，但不知道从哪下手
- 对比两本教材，判断哪本更适合

## 不适用

- 评估考试题目（那是难度/区分度维度）
- 评估 AI 生成的短文（那些本身就该是理解型友好的）
- 评估 OCR 扫描质量（内容正确性问题是前提，不是适合度问题）
